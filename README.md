# 臨床検査値 疾患推定ツール

臨床検査医学スライド（PDF 1〜12）の内容を網羅した、  
検査値の正常/異常判定と疾患候補推定のウェブアプリ。

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動（http://localhost:5173）
npm run dev

# ビルド
npm run build
```

## 機能

- **自動読み取り（AI）**: テキスト貼り付けまたは画像アップロード → Claude APIで検査値を自動抽出・入力
- **検査値入力**: 血球系・凝固系・肝機能・腎機能・内分泌・免疫など全カテゴリをカバー
- **判定カバレッジ分析**:
  - 現在の入力で判定可能な疾患 / あと少しで判定可能 / 判定不可 を表示
  - 追加するとより多くの疾患を判定できる検査値のランキング
  - 陽性候補疾患を否定するために必要な追加検査を表示
- **症候ブランチ**: 検査値異常が確認できた疾患に対し、症候チェックで診断をさらに絞り込み
- **正常範囲一覧**: スライドからの参照値を検索・閲覧

## ファイル構成

```
src/
  data/
    referenceRanges.js  - 全検査項目の正常範囲
    diseases.js         - 疾患定義・判定ロジック
  lib/
    parseLabValues.js   - AI/ルールベース テキスト・画像パーサー
    coverage.js         - カバレッジ分析ユーティリティ
  components/
    AutoInput.jsx       - 自動読み取りUI
    LabInputPanel.jsx   - 検査値入力グリッド
    ResultPanel.jsx     - 判定結果表示
    CoveragePanel.jsx   - カバレッジ・追加検査提案
    ReferenceTable.jsx  - 正常範囲一覧
  App.jsx               - メインアプリ
```

## カバーする疾患カテゴリ

- 貧血（鉄欠乏・巨赤芽球性・溶血性・再生不良性・腎性）
- 出血性疾患（ITP・DIC・血友病A/B・vWD・ワーファリン過剰）
- 肝・胆・膵疾患（急性・慢性肝炎・肝硬変・閉塞性黄疸・NASH・急性膵炎）
- 腎疾患（CKD・ネフローゼ・IgA腎症）
- 代謝疾患（糖尿病・脂質異常症・痛風・メタボリックシンドローム）
- 内分泌疾患（甲状腺機能亢進/低下症・副腎不全・Cushing・先端巨大症・高PRL血症）
- 電解質異常（原発性アルドステロン症・副甲状腺機能亢進/低下症・SIADH）
- 自己免疫疾患（SLE・多発性骨髄腫）
- 感染・炎症（細菌感染症・ウイルス感染症・侵襲性真菌感染症）
- 心疾患（AMI・心不全）
- 腫瘍マーカー（肝細胞癌スクリーニング）
